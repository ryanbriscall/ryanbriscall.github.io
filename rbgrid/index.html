rbGrid 960, 12col

Semi-semantic 960 Grid System

Features

  • 960 pixel.
  • 12 column.
  • Cross-browser compatible.

Why?

Quickly do page layouts.

Semi-semantic?

Not quite semantic, but I naturally wanted to type "one-half" selector name for a half part of the page.
